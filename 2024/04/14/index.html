兜兜转转，最终又回到了个人博客网站～

契机

在如今的社会中，微信已经融入到了生活的方方面面，每个人都与自己的微信已然成为了一体，不可分割。不仅如此，微信也成为了我们工作的重要工具，包括文件传输、工作沟通等等。但也正因如此，自己的微信朋友圈被同事、领导、长辈、亲友关注着，许多事情以及心情已经不能在朋友圈中随意发表了。

借此契机，于是便想搭建一个属于自己的个人博客网站，作为自己的内容输出平台。

搭建

如今的博客框架众多，经过考虑最终选择了 Hexo 作为自己搭建网站的工具。 Hexo 是一个基于 Node.js 的博客框架，支持 Markdown、一键部署、插件和可扩展性等功能。

在部署上，则选择了 Github 的免费仓库作为自己网站的部署平台。由于 GitHub 运行于国外，因此既免去了繁琐的网站备案流程，也省去了一笔购买虚拟服务器的花销。

用途

随着网络的蓬勃发展，如今抖音、快手等短视频平台大量涌入社会，大家大都选择使用短视频形式来记录自己的生活内容，并称之为 Vlog 。但是文字作为文化传承的载体，传承千年至今，我始终认为其有着自己独特的魅力与优势。

因此个人 Blog 网站部署之后，我会通过对知识的吸收、转化、输出这一系列过程，最终形成文字成果记录在Blog上，包括但不限于思想输出、易学笔记、工作经验、日常见闻等等。
